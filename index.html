User object

Name Type Description
id UUID The ID of the User
name String The User's full name

Lock object

Name Type Description
id UUID The ID of the Lock
name String The name of the Lock created by the User
bolt_state String The current state of the bolt (if available)
online_status String The Lock's current online status

Lock Bolt States


Name Description
unknown The current Bolt State is unknown
locked The Lock is locked
unlocked The Lock is unlocked

Lock Online Statuses


Name Description
online The Lock is online and available for remote control
offline The Lock is offline and not available for remote control
